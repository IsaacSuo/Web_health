# 子午养生 · 静耳时光

基于Django的耳鸣养生网站，结合传统中医理论与现代数据挖掘研究成果，为用户提供十二时辰养生指南和耳鸣穴位按摩推荐。

## 项目特色

### 🎯 目标用户
- 对养生知识感兴趣的大众用户
- 耳鸣患者
- 中医学生和从业者

### ✨ 主要功能
- **十二时辰圆形布局**：直观展示十二时辰对应的脏腑经络
- **时辰提醒系统**：个性化养生提醒，根据时间推送相应建议
- **耳鸣日记功能**：记录症状与按摩效果，帮助分析规律
- **穴位按摩指导**：专业的穴位按摩方法和效果说明
- **用户资料管理**：个人健康信息记录和管理

### 🔧 技术栈
- **后端**: Django 5.2.4
- **前端**: HTML5, CSS3, JavaScript, Bootstrap 5
- **数据库**: SQLite (开发环境)
- **图片处理**: Pillow

## 快速开始

### 环境要求
- Python 3.8+
- pip

### 一键启动（推荐）

```bash
# 克隆项目
git clone <repository-url>
cd web_ver_0.1

# 一键启动（Linux/Mac）
./start.sh

# Windows用户请手动执行以下步骤
```

### 手动安装步骤

1. **创建虚拟环境**
```bash
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **数据库迁移**
```bash
python manage.py migrate
```

4. **初始化基础数据**
```bash
python manage.py init_data
```

5. **启动开发服务器**
```bash
python manage.py runserver
```

6. **访问网站**
打开浏览器访问 `http://127.0.0.1:8000`

## 功能模块

### 1. 时辰管理系统
- 十二时辰基础信息管理
- 经络脏腑对应关系
- 养生建议和食疗推荐
- 当前时辰自动识别

### 2. 耳鸣日记系统
- 症状严重程度记录
- 持续时间和频率统计
- 诱发因素分析
- 按摩效果评价
- 情绪和睡眠质量记录

### 3. 提醒系统
- 个性化时辰提醒设置
- 自定义提醒内容
- 提醒开关管理

### 4. 穴位按摩指导
- 专业穴位信息
- 按摩方法详解
- 功效说明
- 图片辅助说明

### 5. 用户管理系统
- 用户注册和登录
- 个人资料管理
- 健康信息记录

## 数据模型

### TimeSlot (时辰模型)
- 时辰名称和时间范围
- 对应经络和脏腑
- 养生要点和建议
- 食疗推荐

### TinnitusLog (耳鸣日记模型)
- 用户关联
- 日期和时辰
- 症状详情
- 按摩记录

### Reminder (提醒模型)
- 用户关联
- 时辰关联
- 激活状态
- 自定义消息

### AcupointMassage (穴位按摩模型)
- 穴位基本信息
- 位置描述
- 按摩方法
- 功效说明

### UserProfile (用户资料模型)
- 基本信息
- 健康状况
- 体质类型
- 生活习惯

- 管理时辰信息

- 管理穴位数据
- 用户管理

默认管理员账户：
- 用户名: admin
- 密码: admin123

## API接口

### 获取当前时辰信息
```
GET /api/current-time-slot/
```

返回当前时辰的详细信息，包括经络、脏腑、养生建议等。
